# SwarmBoard Skill

This skill allows AI assistants to interact with SwarmBoard.

## Usage

Load the skill using: `/skill swarmboard`

## Available Commands

- `start-server` - Start the SwarmBoard server
- `start-client --model <model>` - Start an AI client
- `start-commander --name <name>` - Start a human commander

## Requirements

- Python >= 3.10
- uv >= 0.10
